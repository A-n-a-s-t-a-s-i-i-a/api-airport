# Airport API

This is a REST API for managing an airport system. It provides endpoints for managing users, flights, tickets, and other related information in an airport. The project is built with Django, Django REST Framework, and is intended to be used as a personal project to practice Django, REST APIs, and Python.

## Features

- User authentication and registration.
- Flight management.
- Ticket booking system.
- Admin dashboard for managing users and flights.

## Technologies Used

- Django 5.1
- Django REST Framework
- drf-spectacular for API documentation
- JWT Authentication
- SQLite (or your preferred database)

## Setup Instructions

### Prerequisites

Before you begin, ensure you have the following installed:

- Python 3.12
- pip (Python package installer)
- Django
- Django REST Framework
- drf-spectacular
- SQLite (default, or your preferred database)
